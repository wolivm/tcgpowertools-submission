How to add cards to your inventory

There are four ways to add cards to PowerTools. This guide walks you through each one, so you can pick what fits your situation.

Pick the right mode

From your dashboard, click Add new articles. You'll see four ways to add cards — pick the one that matches what you have in front of you.

If you have…Use
A handful of unsorted singlesSingle Card Mode
A sorted pile from one expansionBy Expansion
A full or near-full expansionBulk
Your inventory already in a CSV or Excel fileImport a File

Before you start: articles vs. copies

PowerTools treats identical cards as one article. An article has a single sell price and one public comment that's shown on supported platforms.

Two cards belong to the same article only if every attribute matches — name, expansion, condition, language, finish, and signed status. Change any one of those, and you get a separate article with its own price and comment.

If you currently use public comments to distinguish copies of the same card (for warehouse location, batch, etc.), use the dedicated Location field in PowerTools instead. Location is private and supports per-copy tracking.

Single Card Mode

Use this when: you have one or more unsorted singles.

  1. From the Add Singles screen, click Single Card Mode.
  2. Start typing the card name in the Card Name field. PowerTools shows matching cards as visual tiles.
  3. Use the arrow keys to pick the right printing, then press Enter to stage it.
  4. Use the hotkeys to set condition, language, and finish without leaving the keyboard. Press Tab to move between the Quantity and Comment fields.
  5. Press Enter or Space to confirm. Press C to stage another copy with the same settings.
  6. When you're done with the batch, click Save at the top of the screen.

By Expansion

Use this when: you have a sorted pile from a single set and want to go down it in order.

  1. From the Add Singles screen, click By Expansion.
  2. Pick a sort order under Sort expansion by — by rarity and collector number, by collector number, by English name, or by rarity and English name. This sets the order you'll move through the cards.
  3. Pick the set under Expansion.
  4. PowerTools shows the first card. Set the quantity, condition, language, and any other attributes — use the hotkeys for speed, and press Tab to move between the Quantity and Comment fields.
  5. Press Space (or Enter) to confirm and advance to the next card.
  6. To jump directly to a specific card, use the collector-number dropdown at the top of the panel.
  7. Click Save when you're done.

Bulk

Use this when: you've got a full or near-full set and want one (or N) of every card as a starting baseline.

  1. From the Add Singles screen, click Bulk.
  2. Pick the set under Expansion. PowerTools shows the rarity breakdown: total cards, plus counts per category (Common, Uncommon, Rare, Mythic, etc.).
  3. Enter how many of each rarity you have. For example, entering 1 in Common, Uncommon, and Rare adds one of every common, uncommon, and rare in the set.
  4. Set the defaults (condition, language, finish, location, comment) — they'll apply to every card in the batch.
  5. Click Add X articles. The workspace populates with the full batch as individual article cards, which you can then edit or delete one by one.
  6. Click Save when you're done.

Import a File

Use this when: your inventory already lives in a CSV or Excel file (for example, an export from Cardmarket).

  1. From the Add Singles screen, click Import a File.
  2. Click Get example CSV file (PowerTools format) in the top-right to download the template.
  3. Fill in the template with your inventory. See the column reference below.
  4. Drag your file into the drop zone, or click to browse.
  5. PowerTools imports the file and stages every row as an article in the workspace.
  6. Review the staged articles, then click Save to push them to your channels.

finishTypeRegular, Foil, etc.

Duplicate rows: If your CSV has multiple rows for identical cards (all attributes match), PowerTools combines them into one article. Quantities stack, and the last price and comment in reading order win.

Once your cards are added, click Save to push them to your connected channels (Cardmarket, Cardtrader, Shopify). Each mode has its own deep-dive guide — links coming soon.
